# AgasthyaKala Website

**Custom Art Studio · Payyannur, Kerala**

A modern, award-winning portfolio website built with Next.js 14, Tailwind CSS, Framer Motion, and Lenis smooth scroll.

---

## 🚀 Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Run locally
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

### 3. Build for production
```bash
npm run build
npm start
```

---

## 🌐 Deploy to Vercel (Free — Recommended)

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Click **Deploy** — done! Your site will be live in 2 minutes

## 📁 Project Structure

```
src/
  app/
    layout.tsx       ← SEO metadata, fonts, schema markup
    page.tsx         ← Main page assembling all sections
    globals.css      ← Global styles + Tailwind
  components/
    SmoothScroll.tsx ← Lenis smooth scroll wrapper
    ScrollProgress.tsx ← Gold progress bar at top
    Navbar.tsx       ← Fixed nav + mobile menu
    Hero.tsx         ← Fullscreen hero with mosaic background
    Marquee.tsx      ← Scrolling text strip
    Gallery.tsx      ← Masonry gallery + lightbox
    About.tsx        ← Studio story + stats
    Process.tsx      ← How to Order (4 steps)
    Commission.tsx   ← WhatsApp commission form
    Contact.tsx      ← Map + contact details
    Footer.tsx       ← Footer with links
public/
  artworks/          ← All 11 artwork images
  logo/              ← Gold Malayalam logo
```
